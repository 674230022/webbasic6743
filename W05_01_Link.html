Workshop: Various Links

ตัวอย่างการใช้งานลิงก์ประเภทต่าง ๆ ในภาษา HTML

1. External Links

External Link คือการเชื่อมโยงไปยังเว็บไซต์ภายนอก โดยใช้ target="_blank"

Visit W3Schools 
<a href="https://www.w3schools.com" target="_blank">
Visit W3Schools
</a>

2. Internal / Anchor Links

ใช้สำหรับเลื่อนไปยังตำแหน่งภายในหน้าเว็บเดียวกัน

<a href="#target-section">
Go to Target Section
</a>
Go to Target Section

Target Section

ตำแหน่งปลายทางของ Anchor Link

3. Email Links

<a href="mailto:student@example.com?subject=Hello">
Send an Email
</a>
Send an Email
ลิงก์นี้จะเปิดโปรแกรมอีเมลของผู้ใช้โดยอัตโนมัติ

4. Download Links

<a href="files/sample.txt" download>
Download File
</a>
Download Text File
ใช้ download เพื่อบังคับให้ไฟล์ถูกดาวน์โหลด

5. Button Links

<a href="https://www.example.com" class="btn green">
Visit Website
</a>
Visit Website
เป็นลิงก์ที่ตกแต่งให้มีลักษณะเหมือนปุ่ม

6. Image Links

<a href="https://www.youtube.com">
<img src="images/youtube.png" alt="YouTube">
</a>
คลิกรูปภาพเพื่อเปิดเว็บไซต์
