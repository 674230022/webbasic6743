Workshop: HTML Text Formatting

ฝึกใช้แท็กรูปแบบขอ้ความ (Formatting Tags) เพื่อสร้างเนื้อหาที่อ่านง่ายและเป็นระเบียบ

1.สิ่งที่ต้องรู้ก่อนเริ่มทำ Workshop

ฝึกใชแ้ท็กรูปแบบขอ้ความ (Formatting Tags) เพื่อสร้างเนื้อหาที่อ่านง่ายและเป็นระเบียบ

  • < b> - ตัวหนา (เน้นเฉย ๆ)
  • < strong> -ตัวหนาแบบ “สา คัญ” เชงิความหมาย (semantic importance)
  • < i> -ตัวหนาแบบสำคัญเชิงความหมาย (semantic importance)
  • < em> -ตัวเอียงแบบเน้นนํ้าเสียง
  • < u> -ขีดเส้นใต้
  • < ins> -ขีดเส้นใต้ ้
  • < mark> -ไฮไลต์ข้อความ
  • < code> -แสดงข้อความเป็นรูปแบบโค้ด เช่น hello
  • < small> -ข้อความขนาดเล็ก
  • < del> -ข้อความขีดฆ่า
  • < s> -ข้อความขีดฆ่า
  • < sub> -ตัวห้อย เช่น H2O
  • < sup> -ตัวยก เช่น x2

    • แท็กที่ไม่รับรองใน HTML

    • < strike> -not support in HTML5
    • < big> -not support in HTML5
    • < tt> -not support in HTML5
    นักเรียนต้องใช้แท็ก formatting อย่างน้อย 5 ชนิดในงานนี้

2.ภารกิจของนักศึกษาที่ต้องทำ

ให้ค้นหาเนื้อหาเกี่ยวกับ “เทคโนโลยีคอมพิวเตอร์สมัยใหม่” โดยต้องมี:

  • หัวข้อ <h1> 1 อัน
  • ย่อหน้า <p> อย่างน้อย 3 ย่อหน้า
  • ใชแ้ท็ก formatting อย่างน้อย 5 ประเภท
  • ข้อความไหนมีการจัด formatting ให้ ใส่สีข้อความเป็นแดงชมพู

3.เทคโนโลนีคอมพิวเตอร์สมัยใหม่

ในปัจจุบัน เทคโนโลยีคอมพิวเตอร์ ได้รับการพัฒนาอย่างต่อเนื่อง ส่งผลให้การทำงานในชีวิตประจำวันมีความสะดวกและรวดเร็วมากขึ้น โดยเฉพาะการนำ ระบบอัตโนมัติ เข้ามาช่วยลดภาระงานของมนุษย์ในหลายด้าน

หนึ่งในเทคโนโลยีที่ได้รับความสนใจคือ ปัญญาประดิษฐ์ (Artificial Intelligence) ซึ่งช่วยให้คอมพิวเตอร์สามารถเรียนรู้ วิเคราะห์ และตอบสนองต่อข้อมูลได้อย่างชาญฉลาด เช่น การแนะนำสินค้า การจดจำใบหน้า และระบบผู้ช่วยอัจฉริยะ

นอกจากนี้ยังมี คลาวด์คอมพิวติ้ง ที่ช่วยให้ผู้ใช้สามารถจัดเก็บข้อมูลผ่านอินเทอร์เน็ต โดยไม่จำเป็นต้องพกพาอุปกรณ์จัดเก็บข้อมูลขนาดใหญ่ รวมถึงการใช้ Big Data เพื่อวิเคราะห์ข้อมูลจำนวนมากในเวลาอันสั้น

ด้านฮาร์ดแวร์เองก็มีการพัฒนาอย่างเห็นได้ชัด เช่น หน่วยประมวลผลแบบ Multi-Core และการออกแบบอุปกรณ์ให้ประหยัดพลังงานมากขึ้น ซึ่งช่วยเพิ่มประสิทธิภาพและลดผลกระทบต่อสิ่งแวดล้อม

ตัวอย่างแนวคิดที่นิยมในปัจจุบัน ได้แก่ Machine Learning ซึ่งเป็นส่วนหนึ่งของ AI และถูกนำไปใช้ในระบบต่าง ๆ อย่างแพร่หลาย
